# Personal Website Project

## Overview
This project is a simple personal website built with HTML, CSS, and JavaScript.  
It includes a homepage and a second page, styled with a shared CSS file, and deployed to a live server.

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API, localStorage)
- Git (version control)
- GitHub (remote repository)
- DigitalOcean Droplet (hosting)
- Nginx (web server)
