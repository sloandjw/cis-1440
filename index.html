Project Overview

This website is developed locally first and continuously deployed to a cloud-hosted Linux server. The process emphasizes version control, reproducibility, and automated content updates.

Development & Deployment Workflow

• Structured the project locally on my PC

• Initialized a Git repository and hosted it on GitHub:

View Repository (CIS-1440)

• Provisioned a virtual machine using DigitalOcean

• Cloned the repository to the server environment

• Configured and deployed the site using an Nginx web server

• Implemented a scheduled script (cron job) to automatically fetch and update NASA's Astronomy Picture of the Day as the site background

• Established a workflow where updates are developed locally and deployed via repository pull
