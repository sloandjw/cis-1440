ArcadeHub

ArcadeHub is a web application created for my CIS-1512 group project. It allows users to choose from three video games. The application is built using Vanilla HTML, CSS, JavaScript and it uses a Node.js (Express) server and a MySQL database to store game information and user data.

Game Jams

I have participated in several game jams, which are events where developers create a game in a short amount of time (usually a few days). I created games using tools and engines including Visual Studio, git, and Unity. You can find my games on my itch.io page.
